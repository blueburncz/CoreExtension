# ce_vec3_clamp_length
`script`
```gml
ce_vec3_clamp_length(v, min, max)
```

## Description
Clamps vector's length between `min` and `max`.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| v | `array` | The vector. |
| min | `real` | The minimum vector length. |
| max | `real` | The maximum vector length. |
