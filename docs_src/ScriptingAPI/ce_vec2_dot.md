# ce_vec2_dot
`script`
```gml
ce_vec2_dot(v1, v2)
```

## Description
Gets the dot product of vectors `v1` and `v2`.

### Arguments
| Name | Type | Description |
| ---- | ---- | ----------- |
| v1 | `array` | The first vector. |
| v2 | `array` | The second vector. |

## Returns
`real` The dot product.
